    Class RoundtripObjectGraphTraversalStrategy

    An implementation of YamlDotNet.Serialization.IObjectGraphTraversalStrategy that traverses properties that are read/write, collections and dictionaries, while ensuring that the graph can be regenerated from the resulting document.
